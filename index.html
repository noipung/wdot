        Wdot

        이미지 팔레트 색 변환기 사이트, Wdot입니다. 아래는 사이트의 사용 방법입니다.

        1. 변환할 이미지를 업로드합니다.
        2. 팔레트의 색상을 선택합니다.
        3. 밝기, 대비, 채도, 디더, 너비, 높이를 원하는 대로 설정합니다.
        4. 다운로드 버튼을 눌러 이미지를 다운로드합니다.
        1. 변환할 픽셀아트 이미지를 업로드합니다.
        2. 왼쪽 상단에서 '원본 표시 중'으로 바꾸면 원본이 표시되며 격자를 보이게 할 수 있는 버튼이 생깁니다. 해당 버튼을 클릭합니다.
        3. 이제 업로드 버튼 옆 '도트 모드'를 킨 상태로, 사이즈를 작게 줄이며 격자가 픽셀에 딱 맞아 떨어질 때까지 줄입니다.
        4. 딱 맞게 되었다면 다시 '도트 표시 중'으로 바꾸면 한 픽셀이 한칸을 차지하는 픽셀아트가 완성된 것을 확인 할 수 있습니다.
